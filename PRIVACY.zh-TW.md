# 隱私權政策

**生效日期：** 2026-03-06

「Tattoo」（以下簡稱本應用程式）為開源的第三方行動應用程式，專為國立臺北科技大學（以下簡稱臺北科大）學生設計，由 NPC（北科程式設計研究社）開發，與臺北科大無隸屬或背書關係。本政策說明本應用程式如何蒐集、處理、利用及保護您的個人資料。

## 一、隱私權保護政策的適用範圍

本政策適用於您使用本應用程式時所涉及的個人資料蒐集、處理、利用與保護。本政策不適用於本應用程式所連結的其他系統或網站（包括臺北科大各項服務系統），當您透過本應用程式連結至其他系統時，適用各該系統的隱私權保護政策。

## 二、個人資料的蒐集、處理及利用方式

當您使用本應用程式時，我們會蒐集、處理及利用以下資料：

### （一）臺北科大帳號密碼

您的學號與密碼用於代您存取臺北科大各項服務。帳號密碼：

- 使用平台安全儲存機制儲存於您的裝置本機（iOS 使用 Keychain，Android 使用 EncryptedSharedPreferences）
- 僅傳送至臺北科大官方伺服器（`ntut.edu.tw`）進行身分驗證
- 絕不會傳送至任何由我們營運的伺服器

### （二）學業與個人資料

通過身分驗證後，本應用程式會從臺北科大系統取得您的課程、成績及個人資訊等學業相關資料。這些資料會快取在您裝置本機的資料庫中，以減少網路請求，不會被傳送至任何外部服務。

### （三）偏好設定

應用程式設定（顯示偏好、主題、語言）儲存於您的裝置本機。

## 三、資料之保護

所有個人資料（帳號密碼、學業紀錄、偏好設定）皆儲存於您的裝置本機。帳號密碼受平台層級加密保護，只有您本人能夠存取。您可以隨時透過清除應用程式資料或解除安裝來刪除所有儲存的資料。

本應用程式不營運任何後端伺服器，不在任何我們控制的伺服器上蒐集、儲存或處理您的資料。

## 四、與第三方共用個人資料之政策

本應用程式絕不會提供、交換、出租或出售任何您的個人資料給其他個人、團體、私人企業或公務機關。

本應用程式使用以下第三方服務：

### （一）Firebase

本應用程式可能包含由 Google 提供的 Firebase Analytics 與 Firebase Crashlytics。啟用時，這些服務會蒐集：

- **Crashlytics：** 當機報告、裝置型號、作業系統版本，以及當機時的應用程式狀態
- **Analytics：** 匿名使用事件（例如畫面瀏覽）與裝置資訊

Firebase 受 [Google 隱私權政策](https://policies.google.com/privacy)規範。

### （二）GitHub

本應用程式會從 GitHub API 取得貢獻者資料，用於「關於」畫面。此請求不包含任何個人資訊。

## 五、對外連結

本應用程式會連結至臺北科大各項服務系統及其他外部網站。該等連結網站不適用本隱私權保護政策，您必須參考各該網站的隱私權保護政策。

## 六、個人資料權利行使方式

依據《個人資料保護法》，您可以行使以下權利：

- 查詢或請求閱覽您的個人資料
- 請求補充或更正
- 請求停止蒐集、處理或利用
- 請求刪除

由於所有資料皆儲存於您的裝置本機，您可以隨時透過清除應用程式資料或解除安裝來行使上述權利。

## 七、關於臺北科大的隱私權實務

您在臺北科大伺服器上的資料受臺北科大自身隱私權政策及臺灣《個人資料保護法》規範。本應用程式僅作為這些服務的客戶端，無法控制臺北科大如何處理您的資料。

## 八、兒童隱私

本應用程式專為大學生設計，不針對 13 歲以下兒童。

## 九、隱私權保護政策之修正

本隱私權保護政策將因應需求隨時進行修正，修正後的條款將提交至本儲存庫，頂部的生效日期將隨之更新。

## 十、聯繫管道

如對本政策有任何問題或疑慮，請至 [github.com/NTUT-NPC/tattoo](https://github.com/NTUT-NPC/tattoo/issues) 提出 issue。
